# flash-gateway

基于spring cloud gateway创建的企业级网关
实现功能如下：</br>
1、http转grpc泛化调用</br>
2、耗时统计</br>
3、全链路日志</br>
4、限流</br>
5、限额</br>
6、跨域设置</br>

持续更新中，原始项目见：https://github.com/Tyoukai/technology-learning/tree/main/springcloud-learning</br>
